# 认知障碍文字生成器Cognitive Impairment Text Generator

这是一个基于网页的文本处理小工具，用于将输入内容转换为带有特殊符号、风格化排版和可复制输出的结果。

## 功能特点

- 支持输入任意文本
- 提供刷新结果与复制结果功能
- 输出区域会展示生成后的内容
- 仅需浏览器即可运行，无需安装额外依赖

## 项目结构

- `Brain Tissue Injury Text Generator.html`：主页面与页面逻辑
- `README.md`：项目说明文档

## 使用方法

1. 打开 `Brain Tissue Injury Text Generator.html`
2. 在输入框中填写或粘贴文本自动生成结果
3. 点击按钮进行刷新、切换语言或复制结果
4. 查看输出区域中的生成内容
