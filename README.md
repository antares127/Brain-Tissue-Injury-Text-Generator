# 脑组织损伤文字生成器

这是一个基于网页的文本处理小工具，用于将输入内容转换为带有特殊符号、风格化排版和可复制输出的结果。

## 功能特点

- 支持输入任意文本
- 可切换显示语言风格
- 提供刷新结果与复制结果功能
- 输出区域会实时展示生成后的内容
- 仅需浏览器即可运行，无需安装额外依赖

## 项目结构

- `index.html`：主页面与页面逻辑
- `README.md`：项目说明文档

## 使用方法

1. 打开 `index.html`
2. 在输入框中填写或粘贴文本
3. 点击按钮进行刷新、切换语言或复制结果
4. 查看输出区域中的生成内容

## 运行方式

可以直接用浏览器打开页面，也可以通过本地开发服务器运行：

```bash
python -m http.server
```

然后访问：

```text
http://localhost:8000
```

## 说明

该项目主要用于演示文本生成效果与前端交互逻辑，适合用于学习 HTML/CSS/JavaScript 的页面开发。
